# Benchmarks

This report is aligned to grading requirements and uses the automated script in `tests/benchmark_ws.py`.

## Metric Definitions

- **Server TTFT**: time from query send over WebSocket to first streamed token (`server_ttft_*`).
- **Connect handshake**: WebSocket connect/open overhead (`connect_handshake_*`).
- **Total latency**: time from query send to terminal stream event (`total_latency_*`).
- **Token rate**: average streamed token throughput (`tokens_per_sec_avg`).
- **Ingest-to-searchable**: upload time to first successful retrieval of injected marker text.

## Environment

- OS: Windows
- Backend: FastAPI + Uvicorn
- Queue: Redis List + `BLPOP` worker consumption
- Worker: Python async worker process
- Vector DB: ChromaDB local persistence
- LLM: Ollama local model (`gemma3:1b`)

## Reproducible Commands

1) Start stack:
- `docker compose up --build -d`

2) Run benchmark suite:
- `python tests/benchmark_ws.py`

3) Inspect machine-readable output:
- `tests/benchmark_results.json`

## Interpretation Notes

- Server TTFT is isolated from connection overhead; report TTFT and handshake separately.
